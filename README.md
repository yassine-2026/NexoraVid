# موقع تحميل الفيديو - النسخة المطورة

هذا الموقع هو أداة متقدمة لتحميل الفيديوهات من يوتيوب ومنصات أخرى، مبني باستخدام Node.js و React.
المميز في هذا المشروع هو قدرته العالية على تخطي حظر يوتيوب (مثل مشكلة "Sign in to confirm you're not a bot") بشكل تلقائي بالكامل دون الحاجة لتدخل المستخدم أو رفع ملفات كوكيز يدوية.

## آلية عمل النظام (3 طبقات حماية ليوتيوب)

عندما يطلب المستخدم تحميل فيديو من يوتيوب، يقوم الخادم بتنفيذ الاستراتيجية التالية:

### الطبقة الأولى: محاكاة أجهزة الجوال عبر yt-dlp
يقوم الخادم بتجربة استخراج بيانات الفيديو باستخدام عميل `ios` الخاص بـ yt-dlp. هذه الطريقة سريعة وفعالة وتتجاوز العديد من القيود المفروضة على متصفحات سطح المكتب العادية.

### الطبقة الثانية: شبكة Piped API
إذا فشلت الطبقة الأولى، ينتقل الخادم فوراً للاستعانة بشبكة من 8 خوادم Piped مختلفة ومستقلة. يقوم الخادم بتجربة هذه المثيلات بالتسلسل حتى ينجح أحدها في جلب البيانات وصيغ التحميل المطلوبة. شبكة Piped أثبتت استقراراً أعلى من Invidious في الفترة الأخيرة.

### الطبقة الثالثة والضمان النهائي: متصفح Playwright الخفي
إذا تم حظر الخوادم الوسيطة، يلجأ النظام لسلاحه الأقوى: محاكاة متصفح كامل (Chromium) يعمل في الخلفية باستخدام `playwright`.
- يفتح الخادم المتصفح.
- يزور صفحة يوتيوب ويتفاعل مع رسالة الموافقة على ملفات تعريف الارتباط (الكوكيز) تلقائياً.
- يسحب ملفات الكوكيز الطازجة ويحولها لصيغة متوافقة (Netscape).
- يمرر هذه الكوكيز إلى yt-dlp للتحميل بنجاح.
- يمسح الكوكيز المؤقتة فور الانتهاء لضمان الأمان وعدم استهلاك المساحة.

## التثبيت والتشغيل
يمكنك تشغيل المشروع بسهولة. تأكد من تثبيت Node.js.

1. قم بتشغيل سكريبت التثبيت:
\`\`\`bash
chmod +x install.sh
./install.sh
\`\`\`
سيقوم هذا السكريبت بتثبيت جميع الحزم المطلوبة، بما في ذلك متصفح Chromium الخاص بـ Playwright اللازم لعمل الطبقة الثالثة.

2. لبدء الخادم في بيئة التطوير:
\`\`\`bash
npm run dev
\`\`\`

3. لبناء ونشر المشروع:
\`\`\`bash
npm run build
npm run start
\`\`\`

تم بناء واجهة المستخدم باستخدام تقنيات حديثة (React & Tailwind CSS)، والخادم باستخدام Express.js.
